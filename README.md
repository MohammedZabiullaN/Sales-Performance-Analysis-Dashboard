# Sales-Performance-Analysis-Dashboard
# Cleaned and analyzed sales data using Advanced Excel to identify business trends and KPIs.
Built an interactive dashboard using Pivot Tables, Pivot Charts, Slicers, and data visualization techniques.
## Project Overview

This project analyzes sales performance across different regions, products, customers, and sales channels using Microsoft Excel.
🎯 Business Problem

Business managers often struggle to analyze large sales datasets manually.

This project was designed to solve that problem by creating an interactive dashboard that answers important business questions such as:

Which products generate the highest sales?
Which region is the most profitable?
Which sales channel performs best?
How are monthly sales changing over time?
Which customers contribute the highest revenue?
🎯 Project Objectives
Build an interactive Excel dashboard.
Analyze sales performance across multiple business dimensions.
Create dynamic reports using Pivot Tables and Charts.
Enable easy filtering using Slicers.
Present business insights visually.
