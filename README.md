# Sales-Performance-Analysis-Dashboard
Cleaned and analyzed sales data using Advanced Excel to identify business trends and KPIs. Built an interactive dashboard using Pivot Tables, Pivot Charts, Slicers, and data visualization techniques.
 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview

The Sales Performance Analysis Dashboard is an interactive business intelligence project developed using Microsoft Excel. The objective of this project is to transform raw sales transaction data into meaningful business insights through dynamic visualizations and key performance indicators (KPIs).

The dashboard enables users to analyze sales performance across different regions, products, customers, and sales channels. By integrating Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Excel formulas, the dashboard provides an efficient way to monitor business performance and support data-driven decision-making.

---

# 🎯 Business Problem

Organizations generate thousands of sales transactions every day, making it difficult to monitor performance manually. This project addresses that challenge by consolidating sales data into an interactive dashboard that highlights revenue trends, profitability, customer behavior, and product performance.

---

# 🎯 Project Objectives

- Compare sales and profit across different regions.

# 🔄 Project Workflow

### 1. Data Preparation
- Cleaned the dataset by checking for missing values and duplicates.
- Standardized column formats and data types.
- Prepared the data for reporting.

### 2. Data Analysis
- Created Pivot Tables to summarize sales and profit.
- Grouped sales by month, region, product, and customer.
- Calculated key business metrics.

### 3. Dashboard Development
Designed an interactive dashboard containing:
- KPI Cards
- Sales Trend Analysis
- Regional Performance
- Product Analysis
- Customer Analysis
- Channel Performance

### 4. Dashboard Interactivity
Implemented slicers for:
- Region
- Product Category
- Sales Channel
- Year
